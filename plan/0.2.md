# 0.2

Goals:
- More advanced language features are well defined and implemented.
- Core library grows beyond basics.
- `more/` has a few key modules for real programs.
- Language tooling includes `audit` and `format` commands and improves in
reliability and ergonomics.

with tests covering the implemented features and protecting against
regressions.

## Language features

- 📝 Advanced polymorphism
  - 📝 virtual types

- 📝 Advanced functions
  - 📝 richer pipe operator with multiple placeholders
  - 📝 placeholders inside more complex expressions
  - 📝 builtins in richer pipe chains
  - 📝 qualified/generic pipe cases with settled precedence
  - 💭 closure model

- 💭 More advanced comptime. keep it minimal and only when it unlocks a
structural compiler/build-system need

- 💭 Threading, concurrency

## Core library
