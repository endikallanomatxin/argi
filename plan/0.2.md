# 0.2

Goals:
- More advanced language features are well defined and implemented.
- Core library grows beyond basics.
- Language tooling includes `audit` and `format` commands and improves in
reliability and ergonomics.

with tests covering the implemented features and protecting against
regressions.

## Language features

- Advanced polymorphism
  - 📝 virtual types

- Advanced functions
  - 📝 richer pipe operator with multiple placeholders
  - 💭 closure model

- 💭 More advanced comptime

- 💭 Threading, concurrency


## Core library

- Strings
  - 💭 better formatting and text helpers

- More collections
  - 💭 sets


## Tooling

- 💭 `build`
  - 💭 incremental compilation
  - 💭 target selection
  - 💭 optimization modes
  - 💭 more explicit output control

- 💭 `lsp`
  - 💭 incremental diagnostics

- 💭 `audit`
  - 💭 checks for language misuse patterns
  - 💭 red flags around capability / reach usage

- 💭 `format`
  - 💭 basic source formatting for supported syntax
