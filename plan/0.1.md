# 0.1

Goals:
- Basic language features are well defined and implemented.
- Core library has basic utilities.
- Language tooling supports `build`, `test`, `lsp` and `init` commands.

with tests covering the implemented features and protecting against
regressions.

## Language features

- 🔧 Types
  - ✅ booleans
  - ✅ integers and floats
  - ✅ characters
  - ✅ aliases
  - ✅ explicit casts
  - ✅ basic type checks
  - ✅ pointers and references
    - ✅ read-only vs read-write pointers
    - ✅ basic dereferencing
    - ✅ reached arguments baseline
  - 🔧 structs
    - ✅ struct declaration
    - ✅ field access
    - 🔧 `init` / `deinit`
    - 🔧 public/private boundary
  - 🔧 choice / nullability / errors
    - ✅ `choice` values with payloads

    - 🔧 error unions

    - 🔧 `unwrap_or`

- 🔧 Variables and constants
  - ✅ local bindings

- 🔧 Functions
  - ✅ function declaration
